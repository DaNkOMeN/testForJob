Программа по переводу средств "Банк"

Добрый день.

Функциональность программы представлена 5 методами

Программу можно запустить 3 способами:1) сборкой мавен проекта 2)java -jar krista-test-1.0-SNAPSHOT-runner.jar

Получение всех клиентов, находящихся внутри БД

localhost:8080/bank/getAll

Добавление нового клиента

localhost:8080/bank/add/{clientName}/{clientPhoneNumber}/{clientPassportNumber}/{clientMoneyNumber}, где clientPhoneNumber

  • clientName - имя клиента.
  • clientPhoneNumber - номер телефона клиента
  • clientPassportNumber - номер паспорта клиента
  • clientMoneyNumber - количество денег клиента

Удаление клиента

localhost:8080/bank/delete/{clientName}/{clientPassportNumber}, где clientPhoneNumber

  • clientName - имя клиента.
  • clientPassportNumber - номер паспорта клиента

Добавление денег клиенту

localhost:8080/bank/addMoney/{clientName}/{clientPassportNumber}/{money},

  • clientName - имя клиента
  • clientPassportNumber - номер паспорта клиента
  • money - количество добавляемых денег

    • Уменьшение денег у клиента

    localhost:8080/bank/removeMoney/{clientName}/{clientPassportNumber}/{money},

    • clientName - имя клиента
    • clientPassportNumber - номер паспорта клиента
    • money - количество убавляемых денег

    Передача денег от одного клиента другому

    localhost:8080/bank/fromFirstToLast/{firstClientName}/{firstClientPassportNumber}/{secondClientName}/{secondClientPassportNumber}/{money},

    • firstClientName - имя клиента, которых передает
    • firstClientPassportNumber - номер паспорта клиента, который передает
    • secondClientName - имя клиента, который получает.
    • secondClientPassportNumber - номер паспорта клиента, который получает
    • money - количество передаваемых денег
